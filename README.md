# Python Data Analysis Mini Project

Bu projede Python ve Pandas kütüphanesi kullanılarak örnek bir veri seti üzerinde
veri temizleme, filtreleme ve temel istatistiksel analizler gerçekleştirilmiştir.

## Kullanılan Teknolojiler
- Python
- Pandas

## Yapılan İşlemler
- DataFrame oluşturma
- Gruplama (GROUP BY mantığı)
- Filtreleme
- Temel analiz çıktıları
